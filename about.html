About She's Here.

"She's here." is an atmospheric, First Person survival game built for Desktop computers (more devices will likely be compatable soon). The game is currently still under development, so please expect glitches / bugs. It is likely the glitches you see are already being fixed for the next build. If your glitch/s is not fixed, please contact support.

You can download the game from the "Download" tab on our website. Be sure to follow the "News" tab for updates and fixes.



Current Build Version: 0.6.7 BETA

Unity Version: 2018 STABLE
