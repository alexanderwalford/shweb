Installation Instructions

Please follow the steps below to install "She's here.". If you require support, please contact the support team using the "Help" page.


Step 1) Extract the folder named "She's here" to your Desktop.

Step 2) Once the folder has been extracted, you can then run the file named "She's here. Official Launcher".

Step 3) The launcher should install and you should be able to play the game!
